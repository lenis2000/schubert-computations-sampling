# Counterexamples to Merzon-Smirnov at n=17,18,19,20

Certification trail for the exact values used in SI Sec. F and in the main
paper's disproof of the Merzon-Smirnov conjecture.

## Reproducing from the repo root

```bash
./schubert --exact --cotrans '<one-line>'
./schubert --exact --transition '<one-line>'
./schubert layered:b1,b2,b3,b4
```

Each successful run writes stdout and stderr to `logs/<name>.log`. The builder
script extracts the exact integer and the wall time from those logs.

## Rivanna helpers

- `rivanna_certify_n19_n20.slurm` reproduces the n=19 transition and n=20
  cotransition/transition certifications on a 48 GB node.
- `rivanna_certify_single.slurm` is the per-job adaptive-memory variant used
  for larger reruns.
- `RIVANNA_RUNBOOK.md` gives a minimal upload / submit / fetch recipe.

## Historical note

`logs/n19_counterex_transition.log` is an aborted local laptop attempt that
contains no final `Result:` line. It is kept as raw audit evidence, but
`build_certificate_csv.py` intentionally ignores it when regenerating
`certificate.csv` and `certificate.tex`.
