# Paper artifacts

This directory holds the raw evidence cited by the SI:

- `counterexamples_n17_20/` contains the exact-certification logs, the
  machine-readable certificate, and the Rivanna rerun helpers for the
  counterexamples at `n = 17, 18, 19, 20`.
- `n100_production/` contains the committed outputs of the `n = 100` MCMC
  production run used for the permuton, liquid-region, and height plots.

See the per-directory `README.md` files for provenance and rerun details.
